# 🎮 Dota 2 Hero Explorer - Build de Produção

Este repositório contém a versão de implantação (build) do projeto **Dota 2 Hero Explorer**. O objetivo desta página é servir como uma vitrine funcional do projeto, demonstrando a integração com a API da Valve/Dota 2 e a interface final ao usuário.

## 🚀 Sobre o Projeto
Uma aplicação web que consome dados em tempo real sobre os heróis do Dota 2, permitindo:
- Listagem completa de heróis.
- Filtragem por atributos principais (Força, Agilidade, Inteligência e Universal).
- Pesquisa dinâmica por nome e builds mais utilizadas.
- Visualização de detalhes específicos de cada herói.
- Formatado para telefones e tablets.

## 🛠️ Tecnologias Utilizadas
- **HTML5 / CSS3** (Design responsivo e temático).
- **JavaScript (ES6+)**.
- **Dota 2 API** (OpenDota / Valve Data).

## 🔒 Segurança e Otimização (Obfustication)
O código JavaScript neste repositório passou por um processo de **ofuscação e minificação**. 
- **Por que?** Em ambientes de produção, é uma prática comum para proteger a propriedade intelectual da lógica de front-end, reduzir o tamanho dos arquivos e dificultar ataques de engenharia reversa.
- **Transparência:** Se você for um recrutador e desejar avaliar a **qualidade, arquitetura e organização do código-fonte original (limpo)**, por favor, entre em contato comigo para solicitar acesso ao repositório privado.
